## 概要
断面シェーダのデモアプリケーションです

## デモ
https://parallax-kk.github.io/CrossSectionShaderDemo/

## 操作方法

|||
|----|----|
|WASDキー|視点の移動|
|EQキー|視点の上下|
|左マウスドラッグ|ギズモを選択して切断面の移動|
|右マウスドラッグ|視点の回転|
|Rキー|視点のリセット|
|Escキー|終了|
